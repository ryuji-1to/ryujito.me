## Technical Stack

The portfolio website is built with a simple yet effective technical stack, including:

- **Next.js**
- **Tailwind CSS**
- **Biome**
  - Linter and Formatter
- **Valibot**
  - validator
- **Shiki**
  - Syntax Highlighter
